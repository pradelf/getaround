# Web service for Getaround app

Bienvenue sur l'interface des web fonctions pour déterminer le prix de location de véhicule sur le projet Get Around de Jedha.

La documentation pour l'API se trouve sous le lien ci-dessous :

* [Web service for Getaround api](https://huggingface.co/spaces/pradelf/getaround-api/)

Il faut ajouter les secrets pour l'authentification auprès d'AWS dans le space de Hugging Face soit AWS_ACCESS_KEY_ID et AWS_SECRET_ACCESS_KEY.
En plaçant ensuite le contenu de ce repertoire comme dépôt d'un space de hugging face, l'image Docker de l'API web sera faite puis le container contenant l'API sera lancée.
